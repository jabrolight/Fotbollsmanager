# Fotbollsmanager
Ett projekt att dokumentera fotbollsligor, genom att registrera nya ligor, lag, se matchhistoriken. Programmet är gjort för föreningar som vill dokumentera sina ligor, för korpenserier att enkelt hålla koll på poäng, mål med mera. 

funktioner: 
* matchregistrering (du regsitrerar hemma och bortalag, vinster, målskillnad, poäng osv uppdateras automatiskt)
* tabellen sorteras utifrån poäng, målskillnad, gjorda mål och till sist alfabetiskt
* matchhistorik sparas i databas (sqlite)
* grafiskt gränssnitt med tkinter FÖR TILLFÄLLET

använda tekniker: 
* Python
* Tkinter
* SQLite/ sqlite3

# Fotbollsmanager
Ett projekt att dokumentera fotbollsligor, genom att registrera nya ligor, lag, se matchhistoriken. Programmet är gjort för föreningar som vill dokumentera sina ligor, för korpenserier att enkelt hålla koll på poäng, mål med mera.

funktioner:
* matchregistrering (du regsitrerar hemma och bortalag, vinster, målskillnad, poäng osv uppdateras automatiskt)
* tabellen sorteras utifrån poäng, målskillnad, gjorda mål och till sist alfabetiskt
* matchhistorik sparas i databas 
* grafiskt gränssnitt med tkinter för tillfället

använda tekniker: 
* Python
* Tkinter
* sqlite3

vad jag har lärt mig
* Att separera backend och frontend
* att använda databaser (SQL)
  
  
plan för framtiden: 
* att kunna skapa, spara och gå tillbaka till egna ligor
* möjlighet att välja mellan liga, utslagning (random lottning), serie + utsalgning
* Flask/ Django för backend
* html/css/js eller react för frontend
* electron/ tauri/ pyinstaller för paketering till .exe

Kör programmet: 
* Klona repor och kör filen main.py
